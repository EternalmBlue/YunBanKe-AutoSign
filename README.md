# 使用方法
cd到autosign.jar文件所在目录，使用java -jar autosign-1.0-SNAPSHOT.jar  启动程序，输入云班课账号密码，即可开始自动签到

# Tip
程序使用java17构建，运行环境也为java17，java17以下版本还未测试。
